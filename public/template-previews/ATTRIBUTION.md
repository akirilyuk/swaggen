# Template preview images

Cover JPEGs in this folder are produced from **Unsplash** and **Pexels** stock photos.

- [Unsplash License](https://unsplash.com/license) — free to use (including commercial); attribution appreciated but not required by the license.
- [Pexels License](https://www.pexels.com/license/) — free to use; attribution appreciated.

Source URLs are listed in `src/lib/templateCoverImageSources.ts`. Regenerate cropped files with:

`pnpm run download:template-covers -- --write`
